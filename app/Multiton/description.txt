Мультитон порождающий шаблон проектирования
который обобщает шаблон одиночка. То есть это Синглтон на стероидах.
В то время как одиночка разрешает лишь создание одного экземпляра класса
мультитон позволяет создавать несколько экземпляров которые управляются
через ассоциативный массив
Создается лишь один экземпляр для каждого из ключей ассоциативного
массива что позволяет контролировать уникальность обьекта по какому либо признаку
