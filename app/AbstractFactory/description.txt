Абстракная фабрика а по другому Интсрументарий по логике больше подходит

Это порождающий паттерн проектирования который позволяет создавать семейства связанных
обьектов не привязываясь к конкретным классам создаваемых обьектов

ссылки на рефакторинг гуру описание этого паттерна и ссылка на сравнение с другими

Семейство обьектов это только одно семейство которые работают между собой
а вот другое семейство не работает с другим создаем только одно семейство
которое работает только со своими родственниками

К примеру создали два семейства и есть переключатель когда мы работаем только с одним
а потом с другим вот тот класс который будет давать доступ к ним порождать и давать тебе
он Абстрактная фабрика

Мы можем переключаться между семействами при условии что оба используют один интерфейс

Определение с вики

Абстрактная фабрика порождающий шаблон проектирования предоставляет интерфейс для создания
семейств взаимосвязанных или взаимозависимых обьектов не специфицируя их конкретных классов.
Шаблон реализуется созданием Абстрактного класса Фактори который предоставляет собой интерфейс
для создания компонентов системы
например для оконного интерфейса он может создавать окна и кнопки
затем пишутся классы реализующие этот интерфейс
